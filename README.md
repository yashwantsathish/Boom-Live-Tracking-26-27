# Basketball Coaching Analytics - Paint Touch Tracker

A live tracking interface for basketball coaching analytics to monitor paint touches per possession during 5v5 practice sessions.

## Quick Start

1. Open `index.html` in any web browser (Chrome, Edge, Firefox)
2. Select lineups for White and Blue teams
3. Track possessions and paint touches during practice
4. Export data to Excel when finished

## Location

## How to Use

### 1. Setup Lineups
- Select 5 players for White team
- Select 5 players for Blue team
- Players show as "#[jersey] [name]"

### 2. Track Possessions
- Click "+1 White Possession" or "+1 Blue Possession"
- Increments possessions for all 5 players on that team

### 3. Log Paint Touches
- Click player buttons in Paint Touch Tracking section
- Works for any player regardless of lineup

### 4. Export to Excel
- **Copy Excel Data**: Copies tab-delimited data - paste directly into Excel with Ctrl+V
- **Export to CSV**: Downloads CSV file you can open in Excel

### 5. Reset Data
- Click "Reset All Data" to start fresh
